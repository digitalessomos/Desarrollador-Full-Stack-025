Proyecto de Tienda Virtual

Este proyecto de tienda virtual está diseñado para ofrecer una experiencia de compra en línea fluida y eficiente. Permite a los consumidores digitales recibir sus compras directamente en la puerta de su casa, con entregas a tiempo y una logística optimizada para atraer y retener a más clientes.

Tecnologías Utilizadas:

  • **Frontend:** Angular
  • **Backend:** API-Firebase
  • **Base de Datos:** Firebase Database

Características Principales:

  • Interfaz de usuario intuitiva y responsive.
  • Gestión de productos y categorías.
  • Proceso de carrito de compras y checkout seguro.
  • Integración con pasarelas de pago.
  • Sistema de seguimiento de pedidos.
  • Panel de administración para gestión de inventario y usuarios.

  • Registro y seguimiento de pedidos.
